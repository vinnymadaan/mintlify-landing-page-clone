# Mintlify Landing Page Clone

This project is a **frontend clone** of the Mintlify landing page built using **HTML and CSS**.  
It was created for learning purposes to practice layout design, responsiveness, and UI structure.

---

## Features

- Clean navigation bar with logo, links, and buttons  
- Hero section with headline, description, and email input  
- Company showcase section  
- Feature explanation sections with modern layout  
- Call-to-action buttons  
- Styled footer with multiple columns  
- Smooth hover effects and glassmorphism UI elements
